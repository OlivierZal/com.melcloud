# MelCloud for Homey

## Introduction
This app integrate `MelCloud` into Homey.

## Usage
1. Install app
2. Add device(s) to Homey.

You can configure the timeout between polls in the app's settings.

## Final note ##
The repository is available at: https://github.com/XattSPT/com.melcloud
