Cette application intègre MELCloud et MELCloud Home dans Homey pour contrôler :

MELCloud (classique) :
- les pompes à chaleur air-air ;
- les pompes à chaleur air-eau ;
- les ventilateurs récupérateurs d'énergie (VRE).

MELCloud Home :
- les pompes à chaleur air-air.
